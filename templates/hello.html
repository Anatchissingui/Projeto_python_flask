Sobre o Projeto

Este aplicativo, desenvolvido em Flask, funciona com base em rotas. Ao inserir "/" após o endereço URL e uma palavra-chave específica, você é encaminhado para uma rota específica.

  • /hello Exibe uma mensagem de "Hello, World!"
  • /login Apresenta informações diferentes dependendo do estado de acesso do usuário.
  • /welcome Apresenta uma mensagem de boas vindas para o usuário que acabou de fazer o login.
